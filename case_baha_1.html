巴哈姆特文章 Web View 改版

隨著用戶需求的變化和技術的進步，巴哈姆特決定對文章 Web View 進行全面的改版，以提升閱讀體驗、增強互動性並統一設計風格。巴哈姆特 APP 的文章頁選擇使用 Web View 進行開發，由於 Web View 可以在不同平台上重複使用相同的代碼，這可以大幅減少開發時間和成本。
相比於需要針對 iOS 和 Android 等多個平台進行原生開發，Web View 支持所見即所得編輯器（WYSIWYG），是一個更快速穩定的選擇。

在 APP 文章頁改版專案中，我負責整體的視覺設計和設計系統的開發，同時也參與了前端切版工作，確保設計理念在實際產品中得到精確呈現。
