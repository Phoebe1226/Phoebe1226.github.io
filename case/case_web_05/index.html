遠一的信念「聽見你的聲音」
適時採取有效策略，更能善盡社會之責，追求公平、和諧的理想

遠一國際法律事務所於2004年8月創立，遠一旗下所有的律師除了法律業務外，也積極參與公益事務，更希望能在快速成長的科技時代，結合所學，為法律注入新的活水，將實務與理論結合、長期在第一線擔任訴訟律師的經驗，以互信互助的合作模式，提供給客戶全方位的服務。

遠一的意涵即為Foresight，取其先見之明、有遠見者一馬當先之意，法律人不應該等到事後解決紛爭，而是應該能預防紛爭發生於其先，遠一不但要比客戶更早發現問題，還要比客戶早一步解決問題。遠一有一群年輕熱情的律師，除了法律，我們比任何人都關心法制環境及企業需求，希望能融入企業佈局，提供客戶更完整的服務。
